contiki-mirror by dak664

Fork of the github contiki mirror with branches for development and testing.

The contiki developers main code repository is at http://contiki.git.sourceforge.net
A github mirror is updated every few days at http:/github.com/contiki/contiki-mirror.
Some links: Contiki's Home page, Online documentation, Wiki, and Bug reporting.

These branches in my fork contain testing or demonstration code:

jackdaw
Additions to menu for RSSI, MAC protocol switching, interface switching (RNDIS, CDC-ECM, CDC-EEM), mass storage mode, sneezer mode. Different USB product ID's are used to prevent Windows from loading the RNDIS driver when the jackdaw is in CDC-ECM mode. Install .INFs with caution, as the wrong one can cause the blue screen of death.

tictactoe
Pass browser strings after a ? to cgi scripts. Sample page here.

sandbox
For dangerous experiments.

dak664
For backing up my uncommitted files.

master
The fork of the github mirror, updated from time to time.

gh-pages
These web pages.
